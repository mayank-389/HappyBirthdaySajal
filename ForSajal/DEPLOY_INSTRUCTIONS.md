# How to Share Your Website with Your Friend

## Method 1: Netlify Drop (Easiest - No Account Needed)

1. **Create a ZIP file:**
   - Select all files in this folder (Ctrl+A)
   - Right-click → "Send to" → "Compressed (zipped) folder"
   - Name it `HappyBirthday.zip`

2. **Deploy:**
   - Go to: https://app.netlify.com/drop
   - Drag and drop your ZIP file
   - Wait 30 seconds for deployment
   - Copy the URL (e.g., `https://random-name-12345.netlify.app`)
   - Share that URL with your friend!

## Method 2: GitHub Pages (Permanent Solution)

1. Create a GitHub account at github.com
2. Create a new repository
3. Upload all files to the repository
4. Go to Settings → Pages
5. Select main branch and save
6. Your site will be live at: `https://yourusername.github.io/repository-name`

## Method 3: Vercel (Also Easy)

1. Go to: https://vercel.com
2. Sign up with GitHub/Google
3. Click "Add New Project"
4. Drag and drop your folder
5. Deploy!

**Recommended: Use Netlify Drop - it's the fastest!**
